# 🐟 OSMU 블로그 생성기 (One Source Multi Use)

본 프로젝트는 하나의 주제로 네이버 블로그, 티스토리, 워드프레스에 최적화된 콘텐츠를 동시 생성하는 인공지능 블로그 마케팅 자동화 도구입니다.

---

## 💎 개발 원칙 (Executive Philosophy)

대표님의 지침에 따라, 본 프로젝트의 모든 기능 개선 및 이슈 해결은 다음의 **[Research-First]** 프로세스를 절대 원칙으로 삼습니다.

1. **[사전 조사]**: 새로운 기능을 구현하거나 오류를 수정하기 전, 반드시 구글 검색 및 최신 기술 트렌드를 조사하여 최적의 사례를 확보한다.
2. **[시행착오 최소화]**: 단순한 가설에 기반한 코딩을 지양하고, 검증된 기술 데이터를 바탕으로 '트라이 앤 에러'를 최소화한다.
3. **[사용자 경험 우선]**: 기술적 화려함보다 네이버 스마트에디터 등 실제 발행 플랫폼과의 완벽한 호환성을 최우선 가치로 둔다.

---

## 🚀 주요 기능
- **원버튼 동시 생성**: Gemini API를 활용한 3개 플랫폼 콘텐츠 실시간 생성
- **Rich-Copy 엔진**: 네이버 스마트에디터 서식(폰트, 크기, 간격) 100% 유지 복사
- **멀티 어투 설정**: 플랫폼별 타겟 독자에 맞춘 개별 어투 적용
- **팩트체크 가이드**: 공공기관 공식 링크 자동 추출 및 보안 검증
