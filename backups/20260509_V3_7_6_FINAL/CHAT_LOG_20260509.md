## 🛠 진행 내역 (세부 작업 내용)

### 1. App.jsx 대규모 리팩토링 (V3.7.6 업데이트)
- **상태(State) 객체 구조 변경**:
  - 기존: `results` 객체 (단일 관리)
  - 변경: `results: { topic: {...}, youtube: {...} }` 형태로 입력 모드(inputMode)에 따른 독립 바구니 생성.
- **팩트체크 리포트 분리**:
  - `groundingMetadata` 역시 단일 상태에서 `{ topic: null, youtube: null }`로 분리하여 각 모드별 구글 검색 근거 데이터를 독립적으로 보존.
- **렌더링 로직 수정**:
  - 모든 JSX 렌더링 코드의 `results[activeTab]` 참조를 `results[inputMode][activeTab]`으로 변경.

### 2. 치명적 오류(WSOD - White Screen of Death) 긴급 진압
- **증상**: 업데이트 후 유튜브 자막 탭에서 'AI 이미지 프롬프트 보기' 버튼 클릭 시 하얀 화면 출력 후 멈춤 현상 발생.
- **원인 파악**: 프롬프트 모달 내부에서 `results[activeTab].section_prompts`를 호출하여 undefined 에러 발생 (기존 참조 방식이 남아있었음).
- **조치 완료**: 해당 구문을 `results[inputMode][activeTab].section_prompts`로 수정하여 버그 완벽 제거 및 핫픽스 배포.
