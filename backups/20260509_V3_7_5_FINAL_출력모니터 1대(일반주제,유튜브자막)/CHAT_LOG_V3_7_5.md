# KODARI BLOG AI V3.7.5 업데이트 요약 및 챗 로그

## 1. 업데이트 내역 (V3.7.5)
- **표 렌더링 엔진 장착 (V3.7.4)**: `remark-gfm`을 도입하여 화면에서 깨지던 표 디자인 완벽 복구.
- **워드프레스 영문 태그 방지 (V3.7.5)**: 시스템 프롬프트를 강화하여 무조건 한국어 해시태그만 출력하도록 통제.
- **컨텍스트 치매 방지 (V3.7.5)**: 유튜브 자막 모드에서 '이 글만 다시 쓰기' 클릭 시 원본 자막을 잃고 비타민 C 같은 엉뚱한 글을 쓰던 치명적인 문맥 상실 버그 해결. 재생성 시에도 유튜브 자막 원본을 AI에게 다시 전달하여 완벽한 재생성 보장.

## 2. 대표님의 긴급 지시 (작업실 분리 프로젝트)
- 대표님의 예리한 지적: "유튜브 자막으로 글을 뽑은 뒤에 일반 주제로 바꾸면 아까 뽑은 글이 날아간다."
- 부장의 진단: 출력 모니터(results state)가 1개라서 발생하는 문제.
- 대표님의 결단: **작업실 완전 분리!** 일반 주제용 모니터 1개, 유튜브 자막용 모니터 1개로 완전히 쪼개서, 탭을 전환해도 각자의 작업물이 유지되도록 대대적인 구조 개조 명령 하달!

## 3. 다음 작전 목표 (V3.7.6 예정)
